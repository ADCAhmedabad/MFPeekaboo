# MFPeekaboo
A demo demonstrating Peek & Pop and Shortcut features with Force Touch. 

--

![](./3DTouch.gif)

OS Version 
--
- iOS 9 (iPhone 6s, iPhone 7 & 7 Plus) 

How to run
--
- Update Xcode to at least 8.0
- Open project xcodeproj
- In Xcode press Cmd-R or choose Run
